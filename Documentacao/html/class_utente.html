Detailed Description

Classe para instanciar utentes.

Esta classe agrupa todas as caracteristicas de um utente. Permite definir os clientes da empresa de transportes e operar sobre alguns dos aspectos mais importantes, como por exemplo: verificar quais as zonas onde o cliente habita e tem escola, verificar se um cliente e utente ou funcionario, encontrar o numero de telemovel do cliente para o poder contactar, etc. Da classe Utente derivam a classe Crianca e a classe Funcionario.
